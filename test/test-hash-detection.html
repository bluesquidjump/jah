4. Mixed Content (JA4 + File Hashes)

This paragraph mixes both types. The JA4 fingerprint t13d1517h2_8daaf6152771_b0da82dd1658 should get a fox icon, while the malware hash SHA256: 275a021bbfb6489e54d471899f7db9d1663fc695ec2fe2a2c4538aabf651fd0f should get a bug icon.

5. Substring Collision Prevention

A SHA256 hash should NOT also match as an MD5 or SHA1. Only ONE icon should appear per hash.
